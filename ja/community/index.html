p5.js コミュニティ声明

p5.jsは、技術を用いた芸術とデザインの創造を探求するコミュニティです。

私たちは、あらゆるジェンダー・セクシャリティ、人種・民族、言語、神経タイプ、体型、能力、階級、宗教、文化、サブカルチャー、政治的見解、年齢、スキルレベル、職業、バックグラウンドの人々からなり、彼らと連帯します。誰もが積極的に参加する時間や財政的余裕、能力を持っているわけではないことを認識しながらも、すべての参加に対して認識と奨励を与えます。アクセスとエンパワーメントを促進・育成します。私たちは、常に学習者です。

私たちは、次のハッシュタグを好んで使用します。#noCodeSnobs（コミュニティを効率よりも大切にするため）、#newKidLove（私たちは皆どこかで初めて始めた）、#unassumeCore（私たちは知識を想定しません）、#BlackLivesMatter（もちろんの理由で）。

実践において:

  • 私たちはコードのエリート主義者ではありません。私たちは知識を仮定したり、誰かが知っていることがあると示唆したりしません。
  • 私たちは、複雑であってもフィードバックのリクエストに積極的に関与することを要求します。
  • 私たちは新規参加者を歓迎し、他者の教育を優先します。私たちは新規参加者と同じ熱意ですべてのタスクに取り組むことを目指しています。なぜなら、専門家と同じくらい新規参加者もこの取り組みで同じ価値があると信じているからです。
  • 私たちは、複数の種類の貢献を積極的に認識し、正当化する努力を継続しています。
  • 私たちは常に助けや指導を提供する用意があります。

紛争時に:

  • 私たちは聞くことに徹します。
  • 私たちは他者の感情を認めながら明確にコミュニケーションをとります。
  • 私たちは自分たちが間違っていると認め、謝罪し、自分たちの行動に責任を負います。
  • 私たちは常に自分自身とコミュニティを改善しようとしています。
  • 私たちはコミュニティを敬意を持ってオープンに保ちます。
  • 私たちは、すべての人が聞かれたと感じるようにします。
  • 私たちは相手に対して思いやりと親切心を持ったやり取りを心がけます。

将来的には:

  • 未来は今です。

貢献する

私たちのコミュニティでは、様々な方法で熱心な人々の協力を求めています。

開発する GitHub は、コードの収集、問題の文書化、コードに関する議論が行われる主要な場所です。 開発チュートリアル をチェックして、はじめましょう。または、自分自身のライブラリを作成してください： あなたのライブラリを作る。

ドキュメント文書化は誰もが好きです。サンプルの移植やドキュメントの追加、チュートリアルの作成など、援助が必要です。

教えるワークショップ、クラス、友人、共同作業者を教えてください！Twitterで@p5xjsをタグ付けし、あなたがやっていることを共有するために最善を尽くします。

作成するp5.jsは、デザイナー、アーティスト、コーダー、プログラマーを探しています。あなたの創造的で素晴らしい作品をフロントページに表示して、他の人々をインスピレーションを与えてください。hello@p5js.orgにご提出ください。

寄付するp5.jsは、アーティストが制作した無料かつオープンソースです。p5.jsの開発を支援するために、Processing財団に寄付してください。
